## Installation

TaskCommander is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TaskCommander'
```

Or only import the Core part:
``` ruby
pod 'TaskCommander/Core'
```
